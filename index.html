Willkommen auf BioQuis

Wie schon Goethe niederschrieb: "Man sieht nur, was man weiß". Ganz in diesem Sinne sollte BioQuis helfen die Lebewesen unserer Umwelt kennen zu lernen und dadurch ermöglichen die überwältigende Vielfalt unseres nächsten Umfeldes wahrzunehmen.

Wie funktioniert diese Seite? (Bildanzeige)

In der Bildanzeige, welche sich hier im Hintergrund befindet, wird ein zufällig ausgewähltes Bild eines Lebewesens aus einer vorgefertigten Liste angezeigt, bei Klick auf das Bild wird der Name und weitere Informationen über das abgebildete Lebewesen angezeigt. Klickt man in dieser Informationsanzeige dann , so wird das Bild in dieser Runde kein weiteres mal wiederholt und das nächste Bild angezeigt. Es empfiehlt sich diesen Button zu wählen wenn man das Lebewesen am Bild richtig erkannt hat. Wählt man so wird das Bild in dieser Runde erneut angezeigt werden.

Du benötigst weitere Hilfe?

Diese Information, weitere Hilfestellungen zur Bildanzeige und weitere Informationen über diese Website können jederzeit durch Klick auf "Hilfe" im Menü am oberen Rand der Website abgerufen werden.
